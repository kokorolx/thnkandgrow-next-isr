Câu hỏi:

```
# ruby
arr =[[1,2], [3,4], [5,6]]
arr.map{|a,b| return a if a + b > 10}

#> LocalJumpError (unexpected return)
Nếu đổi lại thế này
arr.map{|a,b| a if a + b > 10}
#> [nil, nil, 5]
```

Và nếu để vào method như thế này

```
# ruby


def test_arr
  arr =[[1,2], [3,4], [5,6]]
  arr.map{|a, b| return a if a + b > 10}
end

#> 5
```
